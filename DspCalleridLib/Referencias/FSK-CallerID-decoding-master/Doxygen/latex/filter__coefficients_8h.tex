\subsection{Detailed Description}
File containing coefficients for various filters used by the program.

All Coefficients are generated by program \href{http://www-users.cs.york.ac.uk/~fisher/mkfilter/trad.html}{\tt mkfilter}. Corner frequencies can be calculated from \href{http://www.sengpielaudio.com/calculator-cutoffFrequencies.htm}{\tt cut-\/off}. ~\newline


You can design your filter for a particular frequency just by changing the parameters. All the filters used are I\+I\+R and the general equation for it, is

y\mbox{[}n\mbox{]} = c0$\ast$x\mbox{[}n\mbox{]} + c1$\ast$x\mbox{[}n-\/1\mbox{]} + ... + c\+M$\ast$x\mbox{[}n-\/\+M\mbox{]} -\/ ( d1$\ast$y\mbox{[}n-\/1\mbox{]} + d2$\ast$y\mbox{[}n-\/2\mbox{]} + ... + d\+N$\ast$y\mbox{[}n-\/\+N\mbox{]}) where N = no. of previous outputs \& ~\newline
 M = no. of previous inputs ~\newline
 c1,c2,...,c\+M = input coefficients ~\newline
 d1,d2,...,d\+N = output coefficients ~\newline
 x\mbox{[}n\mbox{]} = input values ~\newline
 y\mbox{[}n\mbox{]} = output values ~\newline
